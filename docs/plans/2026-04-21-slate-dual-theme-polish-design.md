# Slate Dual Theme Polish Design

**Goal:** Refine the existing `slate-dual` theme without changing its identity, improving visual hierarchy and selection clarity while keeping the overall tone calm and professional.

## Focus Areas

- make selected states more explicit without turning them into bright blue pills
- separate `tab active` from `list active` so tabs feel slightly lifted
- give child headers a slightly cooler surface identity instead of plain white transparency
- increase the contrast separation between `primary`, `secondary`, and `muted` text
- cool and soften semantic colors slightly so they better match the slate-blue system

## Non-Goals

- no component structure changes
- no new theme ids
- no geometry changes to tabs, lists, or panels

## Success Criteria

- current theme still reads as `Slate Dual`
- active items are easier to scan in dense views
- headers feel more intentionally themed
- long-session readability remains stable
