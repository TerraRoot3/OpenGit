# Slate Dual Overlay Polish Design

**Goal:** Refine `Slate Dual` overlay and floating-surface feedback so menus, dropdowns, and dialog controls feel slightly crisper without changing layout or component structure.

## Focus

- increase hover clarity inside overlays and menus
- make dialog/menu surfaces slightly more intentional and less flat
- keep the overall `Slate Dual` tone restrained and professional

## Scope

- token-only adjustments in `slate-dual`
- no component or spacing changes
- no new theme ids

## Success Criteria

- menu and dropdown hover states are easier to perceive
- floating surfaces feel a bit more separated from page surfaces
- the theme still reads as the same `Slate Dual`
